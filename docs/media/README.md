# Media Placeholders

Put README visuals in this folder using the filenames below.

## Expected Files

- `landing-page.png`
- `landlord-dashboard.png`
- `tenant-dashboard.png`
- `property-flow.gif`
- `payment-flow.gif`

## Tips

- Use 16:9 captures for consistency.
- Keep static screenshots under 400 KB each.
- Keep GIF demos under 8 MB for fast README loading.
